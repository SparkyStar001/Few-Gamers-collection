A collection of simple terminal-based games with voice feedback using `pyttsx3`.  
This project demonstrates Object-Oriented Programming, clean class structure, and interactive voice responses.

## Features
- Text-to-Speech game narration
- Multiple mini-games in one program
- Structured with classes for easy extension
- Simple and beginner-friendly logic

## Games Included
| Game | Description |
|------|-------------|
| Rock Paper Scissors | Play against the computer. |
| Guess The Number | Guess a random number between 1 and 100. |
| Hard Guess | Guess a new random number each turn (more challenging). |
| Cricket | Score runs, but if your number matches the computer's, you're out. |

## How It Works
- The `VoiceAssistant` class handles speaking text aloud.
- Each game is implemented as a class that inherits from the `Game` base class.
- `GameManager` displays the menu and runs selected games.
